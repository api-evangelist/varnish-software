# Varnish SoftwareThis is a repo for managing machine readable artifacts for the [Varnish Software APIs](https://www.varnish-software.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Varnish Software APIs](https://www.varnish-software.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.varnish-software.com/)- [Developer](https://www.varnish-software.com/)- [Twitter](https://twitter.com/varnishsoftware)- [GitHub](https://github.com/varnish)The [APIs.json](https://github.com/api-evangelist/varnish-software/blob/master/apis.json) index provides links to all the relevant links for the Varnish Software APIs, including the OpenAPI and Postman if they are present.
